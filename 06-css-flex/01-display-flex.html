Расположение элементов списка меню в один ряд с помощью flex контейнера

<div class="container">
  <ul class="menu">
    <li class="menu-item"><a href="" class="menu-link">Главная</a></li>
    <li class="menu-item"><a href="" class="menu-link">Новости</a></li>
    <li class="menu-item"><a href="" class="menu-link">О нас</a></li>
    <li class="menu-item"><a href="" class="menu-link">Контакты</a></li>
  </ul>
</div>

Шаг 1

  • Уберем внутренний отступ слева у списка padding-left: 0px;

  • Уберем точки напротив элементов списка list-style: none;

.menu {
  padding-left: 0px;
  list-style: none;
}

Шаг 2

  • Уберем подчеркивание у ссылок text-decoration: none;

  • Установим черный цвет текста ссылок color: #000;

.menu-link {
  text-decoration: none;
  color: #000;
}

Шаг 3

  • Список меню с классом .menu сделаем flex контейнером display: flex;

  padding-left: 0px;
  list-style: none;
  display: flex;

Шаг 4

  • Распределим элементы списка равномерно по flex контейнеру justify-content: space-around;

  padding-left: 0px;
  list-style: none;
  display: flex;
  justify-content: space-around;
}
